TR:

Polinom Regresyon Çalışması (Polynomial Regression)
Bu proje, bağımsız ve bağımlı değişkenler arasındaki ilişkinin doğrusal (lineer) olmadığı durumlarda, verilere en uygun eğriyi uydurarak (curve fitting) tahmin yapmayı amaçlar.

Neler Yapıldı?

Doğrusal Olmayan Analiz: Veri setindeki karmaşık ve eğrisel yapıların tespiti.

Özellik Genişletme: PolynomialFeatures kullanılarak verilerin derecelerinin artırılması ve modelin daha esnek hale getirilmesi.

Pipeline Kullanımı: Veri ölçeklendirme (StandardScaler) ve regresyon adımlarının Pipeline yapısı ile uçtan uca birleştirilmesi.

Model Değerlendirme: R-Kare (R2 Score) ve Hata Kareler Ortalaması (MSE) metrikleri ile modelin tahmin gücünün ölçülmesi.

Görselleştirme: Regresyon eğrisinin veri noktaları üzerindeki uyumunun grafiklerle gösterilmesi.

ENG:

Polynomial Regression Project
This project aims to perform predictive analysis by fitting the most appropriate curve to datasets where the relationship between independent and dependent variables is non-linear.

Key Features:

Non-linear Analysis: Identifying complex and curvilinear patterns within the dataset.

Feature Expansion: Using PolynomialFeatures to increase the degree of independent variables, allowing the model to capture non-linear trends.

Pipeline Implementation: Streamlining the workflow by combining data scaling (StandardScaler) and regression steps through a Pipeline.

Model Evaluation: Measuring predictive accuracy using R-Squared (R2 Score) and Mean Squared Error (MSE) metrics.

Visualization: Plotting the polynomial regression curve against data points to analyze the quality of the fit.
